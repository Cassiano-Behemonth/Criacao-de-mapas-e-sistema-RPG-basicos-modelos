Galeria de Andares

Crie os Setores. Você informará as imagens para as regras habilitadas no passo anterior.

💡 Como funciona o Mapeamento e os IDs (Exemplo Prático): O ID Único (ex: andar_1, cozinha) funciona como o "endereço postal" de cada sala. Ele permite criar caminhos interativos entre seus mapas:
  1. Defina as Salas: Aqui no Passo 3, crie os andares e dê IDs fáceis de lembrar. Exemplo: andar_1 para a Recepção e andar_2 para a Cozinha.
  2. Crie os Atalhos: No Passo 4 (Mapeamento), você estará editando o mapa da Recepção (andar_1). Clique em cima da porta que vai para a Cozinha, digite um nome como "Entrar na Cozinha" e aponte o destino para andar_2.
  3. Teletransporte Automático: No jogo final, quando um jogador clicar nessa porta na Recepção, a tela mudará instantaneamente para a Cozinha, tocando efeitos e abrindo novas pistas!

✓ Dica: Você pode criar quantos andares quiser e interligá-los em qualquer ordem, criando mansões, laboratórios ou cidades inteiras!

📏 Proporção & Resolução Recomendada dos Mapas: Para garantir que o mapa preencha a tela dos jogadores perfeitamente sem distorcer ou deixar grandes bordas vazias, recomendamos usar imagens no formato widescreen **16:9**:
💻 Proporções Ideais (Recomendadas):
  • 16:9 (Widescreen): Ideal para a maioria das TVs e monitores modernos. Resolução sugerida: 1920x1080 ou 1280x720.
  • 16:10 (Notebooks Modernos): Proporção excelente com cobertura vertical estendida. Resolução sugerida: 1920x1200.
🖥️ Outras Proporções Suportadas:
  • 4:3 (Quadrado / Retro): Deixará barras pretas nas laterais em monitores modernos (ex: 1024x768).
  • 21:9 (Ultrawide): Excelente para telas panorâmicas ultra-largas, mas cortará as laterais em notebooks padrão.

Laboratório de IA

O Forge prepara códigos-gatilho avançados sobre medida para você turbinar o export final do mapa.

🚀 Laboratório de Desenvolvimento VTT & IA (Guia Prático)

💻 1. Ferramentas e IDEs Recomendadas:
  • • Antigravity IDE (Pair-Programming): A poderosa ferramenta de codificação agente da Google DeepMind. Ideal para automatizar integrações e programar em par em tempo real com total precisão.
  • • Cursor IDE: Editor de código assistido por IA. Você pode abrir o arquivo exportado, selecionar um trecho com Ctrl+K e pedir para a IA embutida: "Estilize esse modal para parecer um cofre de ferro".
  • • VS Code + Live Server: Instale a extensão "Live Server" no VS Code. Ela permite abrir seu mapa com um clique e atualiza o navegador automaticamente toda vez que você salvar qualquer alteração de código.
🤖 2. Onde gerar os códigos extras:
  • • Claude 3.5 Sonnet: A IA mais recomendada para programar interfaces Web e JavaScript puro. Ela cria estruturas limpas, modais com glassmorphism de ponta e lógicas de puzzles sem erros.
  • • ChatGPT-4o / Gemini 1.5 Pro: Excelentes para debugar erros no console e sugerir links de efeitos de áudio livres de direitos autorais ou comandos de teclado adicionais.
🛠️ Fluxo de Trabalho Passo a Passo:
  1. 1. Clique em "Extrair Código Fonte Final" no Passo 6 para baixar seu arquivo HTML pronto.
  2. 2. Abra o arquivo no VS Code ou Cursor.
  3. 3. Copie um dos prompts gerados abaixo (ex: Cofrinho/Puzzle) e cole na IA (Claude/ChatGPT).
  4. 4. Copie o estilo CSS gerado e cole no cabeçalho do seu HTML, dentro da tag <style>.
  5. 5. Copie o JavaScript (JS) gerado e cole dentro da tag <script> no rodapé do arquivo.
  6. 6. Salve o arquivo e divirta-se jogando e testando localmente com seus amigos!
